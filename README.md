# Educandário Jardim Aeroporto – Website

Website institucional desenvolvido em React para uma escola de educação infantil.

> Projeto em desenvolvimento. Indo para ajustes finais e lançamento 

## Visão Geral

Aplicação frontend moderna, responsiva e acessível, focada em apresentação institucional, comunicação com responsáveis e captação de matrículas.

## Tecnologias

- React 19
- Vite
- React Router
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Lucide React

## Funcionalidades

- Layout totalmente responsivo pra mobile e desktop
- Navegação SPA com React Router
- Integração com WhatsApp
- Formulário de contato
- Galeria de imagens
- Páginas institucionais completas

## Estrutura do Projeto

src/  
├─ components/  
├─ pages/  
├─ assets/  
├─ routes/  
└─ styles/

## Execução Local

### Pré-requisitos
- Node.js 18+
- npm ou pnpm

### Instalação

npm install  
ou  
pnpm install

### Ambiente de Desenvolvimento

npm run dev  
ou  
pnpm run dev  

Acesse: http://localhost:5173

## Build para Produção

npm run build  
ou  
pnpm run build  

Os arquivos finais serão gerados na pasta dist/.
